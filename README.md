# 🕵️‍♀️ Fake Job Posting Detection (EDA Project)

## 📌 Overview

This project focuses on analyzing and understanding patterns in **fake job postings** using **Exploratory Data Analysis (EDA)**. The goal is to uncover insights that help distinguish fraudulent job listings from genuine ones.

---

## 🎯 Objectives

* Perform data cleaning and preprocessing
* Explore key features of job postings
* Identify patterns in fake vs real job listings
* Visualize trends and correlations in the dataset

---

## 📂 Dataset

The dataset contains job postings with features such as:

* Job title
* Location
* Company profile
* Description
* Requirements
* Salary range
* Fraudulent label (target variable)

---

## 🛠️ Tech Stack

* **Python**
* **NumPy**
* **Pandas**
* **Matplotlib**
* **Seaborn**
* **Jupyter Notebook**

---

## 🔍 Key Analysis Performed

* Handling missing values
* Class distribution analysis (Fake vs Real)
* Text-based feature exploration
* Correlation analysis
* Visualization of important attributes

---

## 📊 Insights

* Fake job postings often have **missing or vague information**
* Certain features like **company profile and description length** show strong patterns
* The dataset is **imbalanced** (more real jobs than fake ones)
